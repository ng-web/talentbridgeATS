# Likeslocale Core Extraction Checklist

Use this checklist when preparing reusable elements for extraction into a Likeslocale Core starter repository.

## UI
- [ ] Portal shell isolated
- [ ] Sidebar links standardized
- [ ] Flash banners standardized
- [ ] Stat cards standardized
- [ ] Form pages standardized
- [ ] Status pills standardized
- [ ] Upload panels standardized
- [ ] Empty states standardized

## Backend
- [ ] Role redirect logic reusable
- [ ] Profile completeness service extracted
- [ ] Upload helper service extracted
- [ ] Moderation workflow abstracted
- [ ] Entitlement middleware generalized
- [ ] Notification patterns documented

## Docs
- [ ] Core inventory updated
- [ ] Branding tokens documented
- [ ] Starter repo file map defined
- [ ] Extraction examples added
